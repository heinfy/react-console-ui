# React Console UI

> 基于 `vite` 创建的 `react` + `typescript` 项目，为 [console-server](https://github.com/heinfy/console_server) 提供前端界面。
> 
> UI 界面使用 `ant-design` 组件库构建，界面风格参考 [`ant-design-pro`](https://preview.pro.ant.design/) 。

## 项目概述

1. 使用 `i18next` 实现多语言
2. 使用 `redux` 实现状态管理
3. 使用 `tailwindcss` 实现样式
4. 使用 `echarts` 实现图表
5. 使用 `ant-design` 实现 UI 组件
6. 使用 `eslint` + `prettier`实现代码规范和格式化
7. 使用 `husky` + `commitlint` 实现代码提交规范

## 快速开始

1. 安装依赖

```bash
npm install
```

2. 启动项目

```bash
npm run dev
```

3. 打包项目

```bash
npm run build
```

## 功能介绍

1. 登录
2. 主页
3. 配置管理
4. 日志管理
5. rbac 管理
